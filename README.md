Título: Sistema de Vendas e Gestão de Estoque (Willy Bolsas)

Sobre o Projeto
Este é um sistema de vendas (PDV) e painel administrativo completo, desenvolvido para resolver problemas reais de gestão de loja. O objetivo do sistema é abandonar os controles manuais e automatizar o fluxo de caixa, comissões de vendedores e o controle de estoque em tempo real.

O que já foi construído até agora:

Dashboard (Visão Geral): Um painel inteligente que calcula automaticamente quantas vendas foram feitas no dia, o faturamento total e o valor exato das comissões a serem pagas.

Guardião de Estoque: O sistema tem uma trava de segurança no banco de dados que impede vendas de produtos sem saldo. Além disso, a tela inicial exibe uma tabela de alerta automático, mostrando em vermelho quais produtos precisam de reposição imediata.

Ranking de Performance: Uma área dedicada a mostrar os melhores vendedores do dia, criando um ambiente competitivo e fácil de visualizar.

Terminal de Vendas Moderno: Uma tela de caixa construída com foco em "experiência do usuário" (UX/UI). Utiliza um design em colunas (Grid), separação clara de cores (azul para preços, laranja para troco e destaques) e botões grandes para agilizar o trabalho de quem está vendendo.

Menu Lateral Integrado: Navegação fluida entre a Visão Geral, o Terminal de Vendas e o Estoque, mantendo o usuário sempre focado no sistema.

Tecnologias Utilizadas:

Back-end: Python com o framework Django (responsável por toda a lógica matemática, segurança e banco de dados).

Front-end: HTML5 e CSS3 (responsáveis por todo o design, responsividade e organização visual da tela, sem depender de templates prontos).
